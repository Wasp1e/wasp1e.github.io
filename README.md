 # Проект WiseWallet - это стартап, направленный на создание приложения виртуального помощника для управления личными финансами с использованием искусственного интеллекта.
Чтобы открыть страницу, перейдите по ссылке https://wasp1e.github.io.
Или Установите проект на свой компьютер, затем нажмите на файл index.html и он должен открыться у вас в браузере.
